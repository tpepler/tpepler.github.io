HOWTO: CREATE BITBUCKET REPOSITORIES FOR R PACKAGES

  1. Create a local git repository in RStudio (as an R package).
  2. Create a new git repository on bitbucket.org, such as tpepler/cpc.
  3. In the terminal, change to the directory where the local repository is and use the following commands (adapted for your Bitbucket username and repository name):

    git init
    git remote add origin https://bitbucket.org/tpepler/cpc
    git fetch --all
    git add --all
    git commit -m 'Initial commit'
    git push -u origin master

  4. You will be prompted for your Bitbucket username and password. Once these details are entered, the upload of the local repository to the Bitbucket server will begin.
