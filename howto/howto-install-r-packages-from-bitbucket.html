HOWTO: INSTALL R PACKAGES FROM BITBUCKET REPOSITORIES

  1. In R, install and load the devtools package with

    install.packages('devtools')
    library(devtools)

  2. Install the package from the Bitbucket repository with a command like (for the cpc package hosted at https://bitbucket.org/tpepler/cpc, for example)

    install_bitbucket('tpepler/cpc')

  3. Load the installed package in R with

    library(cpc)

    and it will be ready to use.
