    • Installing the theolib package in R

    1) In R, install and load the devtools package with

    install.packages("devtools")
    library(devtools)

    2) Install the theolib package from the GitHub repository with

    install_github("tpepler/theolib")

    3) Load the installed package in R with

    library(theolib)

    and it will be ready to use!
