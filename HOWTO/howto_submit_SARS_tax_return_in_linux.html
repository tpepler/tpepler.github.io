HOWTO: SUBMIT SARS TAX RETURNS IN UBUNTU LINUX

Online tax return submissions to the South African Revenue Service (SARS) from a Linux machine can be a royal pain, as SARS requires the newest version of Adobe Flash Player to do so. As of the 2016 tax season, the required newest Flash version (11.3) is not supported in Linux anymore. One work-around is to install the Chrome web browser, with its built-in Flash functionality.

  1. Download the Chrome browser and install it on the local computer.
  2. Open Chrome and log on at the SARS e-filing website to view and submit your SARS tax return.
