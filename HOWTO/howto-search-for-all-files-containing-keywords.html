HOWTO: SEARCH FOR ALL FILES CONTAINING CERTAIN KEYWORDS

1) From a terminal, navigate to the directory containing the files of interest.

2) Use

egrep -ail 'keyword1|keyword2|keyword3' *

to find all files (including PDFs) containing either keyword1, keyword2 and/or keyword3; the case is ignored.

3) Alternatively, to find all file names containing a specific word on the entire system, use

locate keyword1
